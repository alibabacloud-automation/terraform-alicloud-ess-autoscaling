# Advanced Example

This example demonstrates advanced ESS Auto Scaling features including multi-AZ deployment, scaling rules, data disks, and internet access.

## Usage

To run this example, execute the following commands:

```bash
terraform init
terraform plan
terraform apply
```

To clean up the resources:

```bash
terraform destroy
```

**Note:** This example will create resources that may incur costs. Make sure to run `terraform destroy` when you are finished to avoid unnecessary charges.
